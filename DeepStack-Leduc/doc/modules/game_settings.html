Module game_settings

Game constants which define the game played by DeepStack.

Fields

suit_count the number of card suits in the deck
rank_count the number of card ranks in the deck
card_count the total number of cards in the deck
board_card_count the number of public cards dealt in the game (revealed after the first betting round)
player_count the number of players in the game
