Module evaluator

Evaluates hand strength in Leduc Hold'em and variants.

Works with hands which contain two or three cards, but assumes that the deck contains no more than two cards of each rank (so three-of-a-kind is not a possible hand).

Hand strength is given as a numerical value, where a lower strength means a stronger hand: high pair < low pair < high card < low card

Functions

evaluate (hand[, impossible_hand_value])
Gives a strength representation for a two or three card hand.

Parameters:

  • hand a vector of two or three cards
  • impossible_hand_value the value to return if the hand is invalid (optional)

Returns:

    the strength value of the hand, or impossible_hand_value if the hand is invalid
batch_eval (board, impossible_hand_value)
Gives strength representations for all private hands on the given board.

Parameters:

  • board a possibly empty vector of board cards
  • impossible_hand_value the value to assign to hands which are invalid on the board

Returns:

    a vector containing a strength value or impossible_hand_value for every private hand
